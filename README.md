# Coeus-Cloud

## Note 开发环境配置

## Project setup 安装
```
yarn install
```

### 本地环境
```
yarn dev
```

### 远程环境
```
yarn remotedev
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
